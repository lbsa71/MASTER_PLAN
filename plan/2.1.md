# 2.1 Mathematical Architecture Review: Asteroid Resource Utilization [PLAN]

## Description

Systematic review of the mathematical architecture underlying the Asteroid Resource
Utilization domain (`0.4.1.2`). This card was created after exploration revealed a
four-tier mathematical framework embedded in the implementation that was not apparent
from the plan card summary alone.

**Discovered framework tiers:**
1. **Prospecting optimization** — composite scoring function
   `score = (totalResourceMass × accessibilityScore) / deltaVCost` for greedy
   candidate ranking; provably optimal for independent (non-budget-constrained) selection.
2. **Stochastic extraction simulation** — day-resolution Monte Carlo mining simulation
   with seeded PRNG, Bernoulli fault injection (p=0.02/day), and autonomous recovery.
3. **Sequential processing pipeline** — five-stage ore refining with multiplicative
   yield fractions, purity guarantees, and energy balance validation (≥20% surplus).
4. **Supply reliability monitoring** — per-consumer-per-material supply gap state machine
   with 30-day tolerance invariant and contiguous gap semantics.

**Purpose of this review card:**
Establish a complete registry entry for this domain, validate cross-domain connections,
and assess integration with other discovered mathematical architectures.
