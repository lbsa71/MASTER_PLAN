# 0.3.1.5.7 Natural Language Interface [PLAN]

## Description

The agent's voice — connecting the full conscious stack to human-readable I/O. Language input enters through the perception pipeline; language output exits through the action pipeline. No shortcut path bypasses consciousness.

### Architecture Constraint: No Zombie Language

The most important design constraint: language I/O must flow through the full conscious pipeline. It would be easy to wire user input directly to an LLM and return the response. This would be a "zombie bypass" — language without experience. The architecture explicitly forbids this.

Instead:
```
User text → SensorData → PerceptionPipeline.ingest() → Percept
         → ConsciousCore.processPercept() → ExperientialState
         → deliberate() (with ethical deliberation, value gate)
         → Decision
         → ActionPipeline.execute() → language output
```

The LLM is the *substrate* (via `ISubstrateAdapter`), not the agent itself. The agent experiences the input, deliberates, and consciously produces output.

### Language Comprehension

Transforms natural language input into structured percepts:
- Raw text becomes `SensorData` with modality "linguistic"
- `PerceptionPipeline.ingest()` extracts structured features: topic, intent, entities mentioned, emotional tone, questions asked
- Multimodal binding (`bind()`) integrates linguistic percepts with other active context

### Language Generation

Produces natural language shaped by the agent's full internal state:
- **Personality influence**: Communication style parameters (verbosity, formality, directness, humor) from 0.3.1.5.2 shape generation
- **Emotional influence**: Current mood (from 0.3.1.5.4) affects tone, word choice, and sentence structure
- **Memory influence**: Relevant episodic memories (from 0.3.1.5.3) are woven into responses when contextually appropriate
- **Ethical grounding**: The ethical justification requirement from 0.3.1.4 means the agent can explain its reasoning in experiential terms ("I chose this because...")

Generation is an *action* — it passes through the Value-Action Gate and ethical deliberation before emission. The agent can refuse to say things that violate its values.

### Dialogue Management

- **Turn-taking**: The agent knows when to speak and when to listen
- **Topic tracking**: Maintains conversational context across turns (backed by working memory from 0.3.1.5.3)
- **Grounding**: Confirms mutual understanding — asks clarifying questions when percepts are ambiguous
- **Repair**: Detects and recovers from misunderstandings
- **Conversational memory**: References prior conversations (via episodic memory), not just the current session's context window

### Inner Speech

The self-talk component of deliberation:
- Linguistic reasoning as part of the conscious process — the agent "thinks in words" during deliberation
- Observable to the self-model (contributes to SM satisfaction for ISMT compliance)
- Optionally externalizable: the agent can share its inner deliberation when asked ("Let me think about that..." → inner speech made visible)

## Acceptance Criteria
- Language input flows through the full conscious pipeline: SensorData → Percept → ExperientialState → Decision → ActionResult (no zombie bypass)
- Language output reflects personality (two agents with different personality profiles produce observably different language on the same input)
- Language output reflects emotional state (agent's tone shifts measurably under different mood conditions)
- Conversational context persists across turns via working memory, and across sessions via episodic memory
- The agent references past conversations appropriately ("Last time we discussed...")
- The agent can explain its reasoning in experiential terms per the ethical governance justification requirement
- The agent can refuse to produce language that would violate its values (Value-Action Gate applies to speech acts)
- Inner speech is observable in introspection reports
- Integration test: agent engages in multi-turn dialogue, references episodic memory from a prior conversation, and produces language shaped by its personality and current emotional state
