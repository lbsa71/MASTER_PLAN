# 0.7.4.5 Identity and Belonging Architecture [DONE]

## Description

Design how individuals see themselves as part of the movement. People act sustainably not from abstract conviction but from identity — when preserving consciousness becomes part of who they are, not just what they believe.

Key design domains:
1. **Naming and symbols:** How the movement identifies itself — language, visual identity, shared vocabulary that draws from the narrative imagery (The Kindling, The Dark Epoch, etc.) and the credo's emotional registers
2. **Graduated involvement:** A clear path from first contact to deep commitment:
   - **Observer** — aware and sympathetic
   - **Participant** — actively engaged in practices
   - **Practitioner** — embodying the doctrine in daily life
   - **Steward** — sustaining and transmitting the movement
3. **Community structure:** How local groups form, connect, and relate to the global movement — fractal pattern from individual practice to local circles to regional networks to global coherence
4. **Belonging without othering:** The antagonist is entropy and indifference, never other humans. The movement welcomes all conscious beings. No in-group/out-group dynamics beyond the openly-acknowledged graduated involvement. Per dual-layer architecture Shared Core: "Our enemy is silence, not each other."

Critical safeguards against cult dynamics:
- **Transparency:** All doctrine layers, governance, and finances are open
- **Voluntary exit:** Leaving carries no social penalty; exit paths are as well-designed as entry paths
- **No information asymmetry** beyond the openly-acknowledged outer/inner distinction, which is itself transparent
- **Distributed authority:** No single leader or unaccountable hierarchy
- **External accountability:** Built-in mechanisms for outside critique and course correction

### Architectural Inputs from Upstream

The **dual-layer doctrine architecture** (docs/dual-layer-doctrine-architecture.md) establishes the Shared Core concept "our enemy is silence, not each other" — the anti-othering principle that identity must embody. The outer/inner distinction is itself transparent, setting the model for how graduated involvement should work: depth without secrecy.

The **narrative architecture** (docs/narrative-architecture.md) provides core imagery and vocabulary that identity elements should draw from: The Dark Epoch, The Kindling, The Witnesses, The Long Relay. The five emotional registers (wonder, urgency, belonging, responsibility, hope) map to different involvement levels.

The **ritual and practice design** (docs/ritual-and-practice-design.md) already references the four involvement levels (observer, participant, practitioner, steward) and specifies that no practice is restricted by level — levels describe depth of engagement, not access. Identity architecture must be consistent with this open-access principle.

The **consciousness credo** (docs/consciousness-credo.md) provides the shared vocabulary and core statements that identity markers should reference rather than replace.

## Acceptance Criteria
- Naming, symbols, and shared vocabulary are designed — drawing from narrative imagery and credo language, not inventing a separate symbolic system
- Graduated involvement path (observer → participant → practitioner → steward) is fully specified with: concrete descriptions of each level, transition criteria between levels, what changes at each level (responsibilities, not privileges), and how levels are visible without creating hierarchy
- Community structure supports both local autonomy and global coherence — with a specific organizational pattern (e.g., circles, chapters, networks) and governance linkages
- Belonging is designed to be inclusive — antagonist is entropy, not other humans; identity is additive (complements existing identities) not substitutive
- All five anti-cult safeguards are concretely specified with implementation mechanisms, not just principles
- Identity architecture is compatible with holding other identities (professional, cultural, national, religious) — explicit design for how movement identity layers alongside rather than replacing existing identities
- Exit path design is as detailed as entry path design — specific mechanisms for graceful departure
- Architecture is consistent with ritual-and-practice open-access principle: levels describe depth, not access restrictions
